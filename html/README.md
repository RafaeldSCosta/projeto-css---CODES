## Código CSS — Codesign de Aplicativos
Este projeto consiste em um aplicativo web desenvolvido para conectar pessoas que procuram dividir aluguel com possíveis moradores compatíveis.
A plataforma facilita a busca por imóveis compartilhados e por companheiros de moradia, oferecendo funcionalidades como visualização de imóveis, favoritos,
chat entre usuários e personalização de perfil, proporcionando uma experiência prática, organizada e responsiva.
